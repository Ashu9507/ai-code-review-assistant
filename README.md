# AI Code Review Assistant

An AI-powered Java code review assistant built using Spring Boot, LangChain4j, and Ollama.

## Features

- Java source code analysis
- AI-powered code review
- Architecture suggestions
- Bug detection
- Edge case identification
- Interview question generation

## Tech Stack

- Java 21
- Spring Boot
- LangChain4j
- Ollama
- Maven

## Architecture

(User → Service → Tools → LLM → Structured Response)
